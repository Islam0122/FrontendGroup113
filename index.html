1Обзор проекта

Islam Checker AI — веб-платформа для автоматической проверки домашних заданий студентов по предметам: HTML, CSS, JavaScript, Python, Английский язык. Система принимает код или текст, анализирует его с помощью ИИ и выдаёт оценку, детальный комментарий, а также определяет — написан ли ответ человеком или сгенерирован ИИ.

Ключевая идея: Преподаватель задаёт условие задания, студент загружает решение — платформа мгновенно выставляет оценку от 0 до 100 баллов и даёт развёрнутый фидбэк на русском языке.
🎯 Для студентов

Быстрая проверка ДЗ, мгновенная оценка и комментарий от ИИ, удобное отслеживание своих результатов.

👩‍🏫 Для преподавателей

Таблица всех сданных работ с фильтрами, определение авторства (человек / ИИ), экономия времени на проверку.

2Цели и задачи

Бизнес-цели

  • Сократить время преподавателя на проверку ДЗ в 5–10 раз
  • Обеспечить мгновенную обратную связь для студентов
  • Создать прозрачную систему оценивания с историей результатов
  • Выявлять работы, написанные ИИ-инструментами (ChatGPT и др.)

Технические задачи

  • Реализовать форму загрузки кода/текста с подсветкой синтаксиса
  • Интегрировать ИИ-модель (GPT-4 / Claude) для анализа и оценивания
  • Разработать систему хранения и фильтрации результатов
  • Обеспечить безопасность данных студентов
  • Поддерживать 5 типов заданий: HTML, CSS, JS, Python, English

3Роли пользователей

🎓 Студент Публичный доступ
  • Заполняет форму: ФИО, группа, тип задания, условие задания
  • Вставляет код или текст решения в редактор
  • Нажимает «Отправить» и видит результат с оценкой и комментарием ИИ
  • Просматривает свои прошлые результаты
👩‍🏫 Преподаватель / Администратор Авторизация
  • Просматривает таблицу всех сданных работ
  • Фильтрует по группе, типу задания, дате, автору (человек/ИИ)
  • Открывает детальную карточку студента с кодом и фидбэком
  • Видит вердикт ИИ: написано человеком или ИИ-инструментом

8Безопасность данных

  • Все данные студентов хранятся в зашифрованной базе данных
  • Доступ к таблице результатов — только через JWT авторизацию
  • API-ключи ИИ-моделей хранятся только на сервере (никогда во frontend)
  • HTTPS-соединение на всех страницах
  • Rate limiting на форму отправки (не более 10 заданий в час с одного IP)
  • Валидация и санитизация всех входных данных на сервере
  • Коды и тексты студентов не передаются третьим лицам

10Нефункциональные требования

⚡ Производительность
  • Время ответа ИИ: не более 15 сек
  • Загрузка страниц: менее 2 сек
  • До 100 одновременных пользователей
📱 Адаптивность
  • Корректная работа на ПК, планшете, телефоне
  • Поддержка браузеров: Chrome, Safari, Firefox
  • Редактор кода адаптирован для мобильных
🌐 Локализация
  • Интерфейс: русский язык
  • Фидбэк от ИИ: русский язык
  • Поддержка кириллицы в полях ввода
🔄 Надёжность
  • Uptime: не менее 99%
  • Резервное копирование БД раз в сутки
  • Graceful fallback при ошибке ИИ API
Замечание по редактору кода: Для HTML-типа задания редактор должен показывать три вкладки (HTML / CSS / JS). Для остальных типов — один редактор. Это соответствует скриншоту с тремя иконками в форме.
